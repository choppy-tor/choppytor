# System Setup and Configuration Files

This directory contains the setup instructions and configuration files used for the experiments.

## Contents

- `setup-steps.md`: Step-by-step instructions for preparing the experimental setup.
- `configs/tor/`: Tor client/relay configuration files.
- `configs/openvpn/`: OpenVPN client/server configuration files.
- `configs/freeswitch/`: FreeSWITCH configuration files used for VoIP calls.
- `configs/pjsip/`: PJSIP caller/callee configuration files.

## Note

Sensitive values such as private keys, passwords, usernames, hostnames, IP addresses, and absolute paths have been removed or anonymized before release.
